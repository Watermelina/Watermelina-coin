How it works

Watermelina is smaller so she fits more easily. The early gaps are larger. The police fruits now use rounded Watermelina-style bodies instead of triangles.
  1. Tap to fart and rise.
  2. Green smoke appears on every jump.
  3. A real local fart sound file plays on every fart.
  4. If Watermelina touches any part of a police fruit, the run ends.
